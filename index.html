About Girl Code

We are Ineke, Katja and Kristin. Girl Code is an initiative founded in 2015 by Ineke at Q42. Now Girl Code is an independent community to inspire girls of all ages to get into code or stay into code. We also aim to raise awareness of the gender gap in tech, so boys are welcome too! We think the world of software engineering can only benefit from having more women on board and creating a more diverse field.

Professional coders aren't necessarily male. That is what we see as an undeniable fact, but also as a statement we want to enforce in the real world. That's why we hope you'll join us in making this difference.

Want to host our next meetup? Interested in speaking or mentoring at any of the following events? Have a topic you think we should discuss or bring to light? Feel free to contact us anytime!
